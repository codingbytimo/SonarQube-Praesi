Was ist SonarQube?

  • Open Source Tool zur statischen Codeanalyse
  • Findet Bugs, Sicherheitslücken, Code Smells
  • Visualisiert Metriken: Komplexität, Coverage, Duplikate
  • Webinterface & CI/CD-Integration

Features

Bugs erkennen

Durch Bugs kann sich ein Programm anders verhalten, als eigentlich gewollt oder gedacht: 

						public int addNumbers (int a, int b) {
							result = a - b;
							return result;
						}

Sicherheitslücken

SonarQube erkennt Sicherheitslücken, bspw. Passwörter, welche im Code hardcoded werden. 

			String username = "admin";
			String password = "%h28JiuWlsL16&gFNnwJ89JHjbd?nj!fs"

Java Showcase

Kleines Maven-Projekt mit fehlerhaftem Code und SonarQube-Analyse

Beispielcode: Java-Klasse


		public class Calculator {
				public int add(int a, int b) {
					return a + b;
				}
				public int divide(int a, int b) {
					return a / b; // Kein Check auf Division durch 0
				}
		}

pom.xml Setup


		  
			org.sonarsource.scanner.maven
			sonar-maven-plugin
			3.9.1.2184

In der Kommandozeile:

mvn clean verify sonar:sonar -Dsonar.login=TOKEN

Analyse-Ergebnis

  • 1 Bug erkannt: mögliche Division durch 0
  • Coverage < 80%: Warnung
  • Code Smell durch fehlende Kommentare
